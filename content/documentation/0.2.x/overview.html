--- title: Overview version: "0.2.x" --- h1. Overview JsSIP is a simple to use JavaScript library which leverages latest developments in "SIP":http://tools.ietf.org/html/rfc3261 and "WebRTC":http://www.webrtc.org to provide a fully featured SIP endpoint in any website. With JsSIP any website can get *Real Time Communications* features using audio, video and more with just a few lines of code. h2. Features * <%= my_lib_link_to "/documentation/misc/sip_websocket" %> transport. * Audio/video calls, instant messaging and presence. * Lightweight! (~140KB). * 100% pure JavaScript built from the ground up. * Easy to use and powerful user API. * Works with OverSIP, Kamailio and Asterisk servers. h2. SIP Standards JsSIP implements the following SIP specifications: * <%= link_to "RFC 3261", "http://tools.ietf.org/html/rfc3261" %> "SIP: Session Initiation Protocol" * <%= link_to "RFC 3326", "http://tools.ietf.org/html/rfc3326" %> "The Reason Header Field for SIP" * <%= link_to "RFC 3327", "http://tools.ietf.org/html/rfc3327" %> "SIP Extension Header Field for Registering Non-Adjacent Contacts" (@Path@ header) * <%= link_to "RFC 3428", "http://tools.ietf.org/html/rfc3428" %> "SIP Extension for Instant Messaging" (@MESSAGE@ method) * <%= link_to "RFC 5626", "http://tools.ietf.org/html/rfc5626" %> "Managing Client-Initiated Connections in SIP" (@Outbound@ mechanism) * <%= link_to "RFC 5954", "http://tools.ietf.org/html/rfc5954" %> "Essential Correction for IPv6 ABNF and URI Comparison in RFC 3261" * <%= link_to "RFC 6026", "http://tools.ietf.org/html/rfc6026" %> "Correct Transaction Handling for 2xx Responses to SIP INVITE Requests" * <%= link_to "draft-ietf-sipcore-sip-websocket", "http://tools.ietf.org/html/draft-ietf-sipcore-sip-websocket" %> "The WebSocket Protocol as a Transport for SIP"
